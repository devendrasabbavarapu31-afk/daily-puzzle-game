# Daily Puzzle Logic Game

A full-stack Daily Puzzle application built using React and Flask.

## Tech Stack
- Frontend: ReactJS
- Backend: Flask
- Database: SQLite

## Features
- Daily logic puzzle
- Answer validation
- Hint system
- Optimized API usage

## How to Run

### Backend
cd backend  
pip install -r requirements.txt  
python app.py  

### Frontend
cd frontend  
npm install  
npm start  

## Performance & Security
- Cached daily puzzles
- Server-side validation
- Minimal API calls
